Who am I?

I am a Computer Science student at VŠB – Technical University of Ostrava. I am a student who wants to work on interesting projects with friendly and inspiring humans. I like trying new things and being creative (not only in IT). I love peace and also some risk at the same time. And I am driven by interest, not by templates.

What can I do?

I can work with many tools. For writing code I am usually using VS Code. In past I used Sublime Text or I tried and worked a little bit in Notepad++, MS Visual Studio and IntelliJ IDEA Community.

For documentation I can use of course office software, but also Notion, OneNote, Rnote, AutoCAD or drawio for taking notes, making design and for some planning.

I can work on junior level programming languages: PHP (Not much really), C, GDScript. Currently I am learning: Assembly, C++, TypeScript (just a little bit). (No, HTML and CSS are not programming languages and that's why they are not there. But I can work with them.)

I have some experience with video and photo editing too. I tried many video editors, but in the end I ended in DaVinci Resolve. And in photo editors I worked a lot with Zoner Studio.

I can't forget. I also can understand hardware in some ways so I can build (put components together) personal computer or just select some other additional hardware. That means I can make documentation, simple websites or just make simple Linux and Windows programs. And primarily I am using Linux specifically Fedora KDE spin. And of course I can work with Windows too. And Docker a little bit.

SilverTool Prototyp

SilverTool Prototype is a small project using C++ with the Qt framework. This application can read HTML documents via the Chromium core. The primary purpose of this application is to keep custom documentation in one place, which helps me stop searching for the same thing again and again. It's also useful for other things, like storing standard structures just to copy and paste.


See on GitHub

TowerDefense and CatJumper

TowerDefense is a game made in the C language with the SDL2 library as a first-semester project for the UPR course at VŠB. The code is modular (or at least supposed to be) and every texture (I know they could be better) was made by me. Audio is not included.
CatJumper is a simple 2D platform game made in the Godot engine, specifically using GDScript. And no, I did not make any of the textures for this one.
