Projects TowerDefence & CatJumper

TowerDefence

What is it about

TowerDefence is a simple and also my first game made via SDL2 library in C. It features one tower type, a map, four types of enemies, a coin system, and a speed-scaling system. The game saves your high score and it is possible to win. Every enemy gives you a different amount of coins. Towers can only be placed on specific spots if you have enough coins, and they can also be upgraded. Enemy spawn rates, health, and speed increase every time you complete a wave. The game has only one map and three menus: the Main Menu, Victory Menu, and Game Over Menu.

How it is made

The project structure isn't perfect, mainly because every function is in a header file. The main file contains only the standard logic, like window setup and background rendering. However, I did my best to make the game modular, so adding new functions is quite easy. Also, the game has VSync, so you don't have to worry about the game running at over 1000 FPS.

CatJumper

What is it about

CatJumper is a simple game made with the Godot Engine. The game has a simple menu and map, featuring a lighting mechanic. It is a platformer game, so the player can move left and right, jump, and fall. It has a soundtrack and various sound effects. There are three types of coins, and every coin gives you a different amount of score. Also, the game has simple audio settings. If you complete a level, your score will be saved if it is better than your previous high score. You can also move around in the main menu. The game features five menus: Main Menu, Settings, Level Selector, Level Completion, and Pause Menu.

How it is made

None of the assets are mine. I downloaded every asset (if I remember correctly) from itch.io as a free version. The game has adaptive VSync to set an FPS limit. This project is older than Tower Defence, and at that time, I didn't know how to make the game as modular as I can now. Not gonna lie, the code is a bit of a spaghetti mess.
